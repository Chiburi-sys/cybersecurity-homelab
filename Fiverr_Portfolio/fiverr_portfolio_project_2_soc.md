## 2. Project Description (Copy/Paste this into the About text)

> ### Professional SOC Documentation & Incident Triage
>
> Engineered a comprehensive suite of Security Operations Center (SOC) documentation, including standardized Incident Response (IR) playbooks, alert triage logs, and detailed escalation matrices.
>
> **Key Documentation Capabilities:**
>
> * **Standardized Incident Reports:** Authored detailed, actionable incident reports for simulated Active Directory password spraying and SMB brute-force attacks, mapping all findings to the MITRE ATT&CK framework.
> * **Custom IR Playbooks:** Developed step-by-step mitigation playbooks for Tier-1 and Tier-2 analysts, covering threat containment, eradication, and post-incident recovery for high-severity alerts (e.g., Kerberoasting, Suspicious Process Execution).
> * **Escalation Matrices:** Designed clear, SLA-driven escalation workflows to reduce Mean Time to Resolve (MTTR) and ensure smooth, contextual handoffs between SOC tiers during active breaches.
>
> This portfolio demonstrates the ability to translate complex, raw security events into repeatable, human-readable processes that strengthen an organization's overall defensive posture.
