## How to Export these to PDF for Fiverr

You can easily convert these Markdown files to PDF using any of your standard markdown tools:

### Option 1: Using your Markdown Editor (Recommended)

If you use **VS Code**, **Obsidian**, or **Typora** to write your lab documentation:

1. Open the `.md` file in the editor.
2. Look for the "Export to PDF", "Print to PDF", or "Save as PDF" option in the File menu or command palette.

### Option 2: Using your Web Browser

1. Open the `.md` file in a Markdown viewer extension in Firefox/Chrome.
2. Press `Ctrl + P` and select "Save to PDF".
