# Resume & application materials

| File | Use |
|------|-----|
| `Stacy_Bostick_Resume.md` | Primary resume in Markdown (export to PDF/Word as needed) |
| `resume_cover_letter_blurbs.md` | Short summaries for applications and interviews |
| `Stacy_Bostick_Cover_Letter.md`, `Stacy_Bostick_Cover_Letter_General.md` | Cover letter templates |
| `extract.py` | Optional helper: prints text from local `.docx` files placed next to this script (Word sources are not stored in the repo) |
