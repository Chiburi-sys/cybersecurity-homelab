# Lab Network Configuration

This folder documents the architecture of the internal cybersecurity lab network, including VM networking, static IP assignments, and VirtualBox configuration.

## Contents
- network_diagram.md — Visual overview of the lab network
- vm_ip_map.md — Static IP assignments for each VM
- virtualbox_interface.png — VirtualBox network settings reference

## Goal
Provide a clear blueprint of the lab environment for reproducibility and troubleshooting.
