# Splunk SIEM Setup and Detection Engineering

This folder documents the installation, configuration, and use of Splunk as a SIEM in the home lab. It includes log ingestion, Sysmon setup, and SPL queries for detection.

## Contents
- splunk_install_ubuntu.md — Installation steps on Ubuntu Server
- sysmon_config_windows.md — Sysmon setup and configuration
- log_forwarding_setup.md — Universal Forwarder configuration
- screenshots/ — Dashboards, alerts, and search examples

## Goal
Demonstrate SIEM deployment, log ingestion, and detection engineering skills using Splunk.
