# SIEM Deployment & Detection Engineering

This folder documents the installation, configuration, and use of both Splunk and Wazuh as SIEM solutions in the home lab. It includes log ingestion, Sysmon setup, and SPL/Wazuh query rules for detection.

## Contents

- splunk_install_ubuntu.md — Installation steps on Ubuntu Server
- wazuh_setup.md — Docker-based deployment of Wazuh on Garuda Linux
- sysmon_config_windows.md — Sysmon setup and configuration
- log_forwarding_setup.md — Universal Forwarder configuration for Splunk
- spl_queries.md — Splunk detection queries
- wazuh_screenshot_guide.md — Generating realistic alerts in Wazuh
- screenshots/ — Dashboards, alerts, and search examples

## Goal

Demonstrate dual-SIEM deployment, log ingestion, alert configuration, and detection engineering skills using both Splunk and Wazuh.
