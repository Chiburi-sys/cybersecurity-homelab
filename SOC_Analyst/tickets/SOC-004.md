---

## Summary

Alert fired for PowerShell execution on Win11 workstation with `-nop -exec bypass` flags. Investigation determined this was a scheduled Sysmon configuration update executed by the SYSTEM account via `services.exe`.

---

## Detection

- **Alert:** Suspicious Process Execution Detected
- **Source:** Splunk (Sysmon EventCode 1 — Process Creation)
- **Trigger:** PowerShell with execution policy bypass flags

---

## Investigation

### Step 1: Review Process Details

```spl
index=main source="WinEventLog:Microsoft-Windows-Sysmon/Operational" EventCode=1
  host="WINDOWS" Image="*powershell*"
| table _time, User, Image, CommandLine, ParentImage, ParentCommandLine
| head 5
```

**Process Details:**

| Field | Value |
|-------|-------|
| User | NT AUTHORITY\SYSTEM |
| Image | C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe |
| CommandLine | `powershell.exe -nop -exec bypass -file C:\Windows\Temp\sysmon_update.ps1` |
| Parent Process | C:\Windows\System32\services.exe |

### Step 2: Assess Legitimacy

| Indicator | Value | Assessment |
|-----------|-------|------------|
| User | SYSTEM | ✅ Expected for scheduled tasks |
| Parent | services.exe | ✅ Standard Windows service manager |
| Script path | C:\Windows\Temp\ | ⚠️ Temp folder, but acceptable for updates |
| Time | 10:32 AM (business hours) | ✅ Normal |

### Step 3: Check for Network Activity

```spl
index=main source="WinEventLog:Microsoft-Windows-Sysmon/Operational" EventCode=3
  Image="*powershell*" host="WINDOWS"
| table _time, DestinationIp, DestinationPort
```

**Result:** No outbound network connections from this PowerShell instance. No C2 indicators.

---

## Actions Taken

- [x] Verified parent process (services.exe — legitimate)
- [x] Confirmed no outbound network activity
- [x] Confirmed Sysmon update script is expected
- [x] Closed as false positive

## Tuning Recommendation

Add exclusion rule to the "Suspicious Process" alert:

```spl
NOT (ParentImage="*services.exe" AND CommandLine="*sysmon_update*")
```

This reduces alert fatigue from known maintenance scripts.

---

## Resolution

**Closed — False Positive.** Sysmon configuration update via scheduled task. Tuning recommendation submitted.
