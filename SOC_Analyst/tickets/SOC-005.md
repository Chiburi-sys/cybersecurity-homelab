## Summary

A password spray attack was detected targeting 5 domain accounts on cybersec.local. The attacker used CrackMapExec to test a single password ("Password123!") against multiple accounts via SMB. The `j.smith` account was compromised due to a weak password.

---

## Detection

- **Alert:** Brute Force — Multiple Failed Logins
- **Source:** Splunk (EventCode 4625)
- **Trigger:** 10 failed logins from 192.168.50.30 within 2 minutes

---

## Investigation

### Step 1: Identify the Attack Pattern

```spl
index=main EventCode=4625 Source_Network_Address="192.168.50.30"
| table _time, Account_Name, Failure_Reason
| sort _time
```

**Result:** Same password tested against j.smith, j.doe, b.wilson, svc_sql, s.bostick in rapid succession — classic spray pattern.

### Step 2: Identify Compromised Accounts

```spl
index=main (EventCode=4625 OR EventCode=4624) Source_Network_Address="192.168.50.30"
| stats values(EventCode) as EventCodes, count by Account_Name
| sort -count
```

| Account | Failed (4625) | Success (4624) | Status |
|---------|--------------|----------------|--------|
| j.smith | 2 | 2 | ⚠️ **COMPROMISED** |
| j.doe | 2 | 0 | Safe |
| b.wilson | 2 | 0 | Safe |
| svc_sql | 2 | 0 | Safe |
| s.bostick | 2 | 0 | Safe |

### Step 3: Assess Impact

```spl
index=main EventCode=4624 Account_Name="j.smith" Source_Network_Address="192.168.50.30"
| table _time, Logon_Type, Workstation_Name
```

**Result:** j.smith authenticated via SMB (Logon Type 3) from the attacker machine. Account used for subsequent domain enumeration.

## Actions Taken

- [x] Confirmed password spray pattern (1 password × 5 accounts)
- [x] Identified j.smith as the only compromised account
- [x] Traced post-compromise activity (enumeration, Kerberoasting)
- [x] Correlated with SOC-001 and SOC-002

## Containment Steps

- [x] Recommended immediate password reset for j.smith
- [x] Recommended account lockout policy implementation
- [x] Documented full attack chain in incident report (IR-002)

---

## Root Cause

The `j.smith` account used a weak, commonly guessed password ("Password123!"). No account lockout policy was configured, allowing unlimited password attempts.

---

## Resolution

**Closed** — Attack documented, containment recommendations provided. Full incident report available at `Incident_Reports/002_ad_password_spray.md`.
